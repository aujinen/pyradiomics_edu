## このフォルダ内の「教育用コンテンツ」について
このフォルダ内のコンテンツは教育目的での利用を前提としているため、臨床での利用はしないで下さい。<br>
Radiomicの各種計算過程は複雑ですが、手順を追えば難しくはなく、学生自らが計算手順を追体験しつつ、<br>
どういった目的で使えるのかといった理解の補助になるよう意識して作成していきます。

## About the "Educational Content" in this folder
The content in this folder is intended solely for educational purposes; please do not use it in a clinical setting.<br>
While the various calculation processes involved in radiomics are complex, they are not difficult to follow step-by-step.<br>
These materials have been designed to help students gain a practical understanding of the procedures and the purposes for which these methods can be applied.
